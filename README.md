# Zero Jekyll Theme

Zero a minimal two-column Jekyll theme based on bootstrap 3.

## Features

- Tags & Categories
- SideBar 
- Table of Contents (support [Scrollspy](https://getbootstrap.com/docs/3.3/javascript/#scrollspy))
- Latex support (MathJax)
- Social icons
- Code block highlighting
- Mobile display

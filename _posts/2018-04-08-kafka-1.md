# 下载地址:

github pro:[kafka docker](https://github.com/wurstmeister/kafka-docker)

# Usage:

```

Start a cluster:

docker-compose up -d
Add more brokers:

docker-compose scale kafka=3
Destroy a cluster:

docker-compose stop

```
